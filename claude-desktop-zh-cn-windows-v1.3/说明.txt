Claude Windows 简体中文汉化工具

当前版本：

1.3 翻译优化版。
安装方式仍沿用固定路径复制版，不直接修改原版 Claude。
本版合并了更自然的设置页、连接器、遥测、第三方推理配置等中文表达，并增加硬编码界面文本替换。

使用方法：

1. 关闭 Claude。
2. 双击“一键安装汉化.cmd”。
3. 如果 Windows 请求管理员权限，请选择允许。
4. 等待脚本完成。
5. 以后从桌面或开始菜单里的 Claude zh-CN 快捷方式启动。

Cowork 兼容：

如果原版 Claude 目录不可写，脚本会把汉化副本固定安装到：

%LOCALAPPDATA%\Programs\Claude-zh-CN

实际运行目录里会包含 WindowsApps 路径段，用于兼容 Cowork。
不要移动或改名 Claude-zh-CN 里面的 WindowsApps、Claude_版本_x64__pzs8sxrjxfjjc、app 文件夹。

备份：

脚本会在当前工具文件夹里创建“Claude备份”，只保留最新一份。
同一个 Claude 目录和同一个版本重复运行时，会复用已有备份，不会每次都重新复制大文件。
语言包、手动修改翻译.json、硬编码替换表和 Claude 版本都没变时，还会复用 Web 汉化缓存，重复运行通常只需要几秒。
这个备份不是运行 Claude 必需的，只是出问题时用于恢复参考。

恢复英文：

双击“恢复英文.cmd”。

手动修改翻译：

打开“手动修改翻译.json”，搜索你在 Claude 界面看到的中文，只修改冒号右边双引号里的中文。
如果想参考英文原文，请打开“翻译对照表.json”。
保存后重新运行“一键安装汉化.cmd”。

安装完成后：

如果 Claude 已经能正常显示中文，可以删除这个汉化工具文件夹。
删除后中文 Claude 仍然可以继续使用。
如果以后还想继续改翻译或保留备份，请先不要删除。
不要删除 %LOCALAPPDATA%\Programs\Claude-zh-CN 文件夹，它是实际运行的汉化版 Claude。

Cowork 工作区权限修复：

如果 Cowork 报 HCS 0x80070005、rootfs.vhdx 拒绝访问，原版 Claude 也可能同样报错。
这通常是 Windows 虚拟机账户没有权限读取 Claude 工作区虚拟磁盘。
关闭 Claude 后，运行“修复Cowork工作区权限.cmd”，再重新打开 Claude。

原版 Claude：

不建议卸载原版 Claude。
卸载后，已经复制出来的汉化版通常仍可通过 Claude zh-CN 快捷方式启动，但官方更新、重新复制新版和部分系统入口可能受影响。
如果后续出问题，先重新安装原版 Claude，再运行“一键安装汉化.cmd”。

不要修改 translation-memory.json。
不要把“翻译对照表.json”当作安装文件修改，它只是参考英文原文。
不要删除“程序文件-不要改名”文件夹。
