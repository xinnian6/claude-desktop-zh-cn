Claude 简体中文汉化工具 1.3 翻译优化记录

本版保留 1.2 的安装方式：

1. 不直接修改原版 Claude。
2. 汉化副本固定安装到 %LOCALAPPDATA%\Programs\Claude-zh-CN。
3. 保留 WindowsApps 路径段以兼容 Cowork。
4. 桌面只创建 Claude zh-CN 快捷方式。

本版主要优化：

1. 合并其他开源汉化项目中更自然的中文表达。
2. 自动修正 661 条明显英文残留或中英硬拼的翻译。
3. 人工优先修正 5 条常见设置页翻译。
4. 新增 435 条硬编码界面文本替换，覆盖普通语言 JSON 不容易覆盖的设置页标题、侧边栏按钮、遥测说明等文本。

保留的关键修复：

1. Language 语言列表中 zh-CN 会放到第一位。
2. 网络错误提示保留中文：请检查你的网络连接、VPN 或代理设置，然后重试。
3. Cowork 兼容逻辑不改为直接修改原版 Claude。
